# Contributing

Thanks for your interest in contributing to GitHub Menu Bar!

## Requirements

- macOS 14.0+
- Xcode 16+ or Swift 6.0+
- GitHub CLI (`gh`)

## Reporting Issues

When reporting bugs, please include:

- macOS version
- Steps to reproduce
- Expected vs actual behavior
- Output of `gh auth status`
